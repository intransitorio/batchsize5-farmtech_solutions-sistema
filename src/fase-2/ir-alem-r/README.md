# Ir Além 2 - Análise em R

## Objetivo
Aplicar uma análise estatística simples em R para complementar a decisão de irrigação do sistema da Fase 2.

## Arquivo principal
- `analise_irrigacao.R`

## O que o script faz
- lê um CSV de exemplo;
- calcula média e desvio padrão da umidade;
- calcula média de pH;
- gera uma recomendação simples de irrigação.
